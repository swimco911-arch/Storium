# 🛸 STORIUM

> *A digital bookshelf where stories drift through space.*

**Storium**은 스토리북을 아름다운 3D 책장에 전시하고, 필터·검색·통계로 탐색할 수 있는 온라인 서재입니다.  
현재는 Google Sheets CSV를 데이터 소스로 사용하며, 순수 HTML/CSS/JS로 동작해 별도 서버 없이 GitHub Pages나 Vercel에 바로 배포할 수 있습니다.

---

## ✨ 주요 기능

### 📚 3D 책장 (Shelf)
- 실제 나무 책장처럼 표현된 입체 UI — 테두리 명암, 그림자, 책등 조명까지 재현
- 책마다 고유 색상이 자동 배정되며, 표지 이미지가 있으면 실제 커버로 표시
- 완독 리본 🔴 · 찜 리본 🔖 이 책 모서리에 표시되어 한눈에 상태 파악 가능
- 신작 배지(`NEW`) — 등록일 기준 30일 이내 자동 표시

### 🌟 추천 컬렉션 (Featured Board)
- `recommend = 1` 로 표시된 책만 별도 진열대에 큰 표지로 노출
- 접기/펼치기 가능한 패널 구조
- 필터 적용 시 추천 컬렉션도 함께 반응

### 🔍 검색 & 필터
| 필터 | 설명 |
|---|---|
| **장르** | 판타지·로맨스·SF 등 카테고리 필터 |
| **대상 연령** | 유아·아동·전체 등 연령 태그 |
| **읽음 상태** | 전체 / 읽음 / 미읽음 |
| **정렬** | 최신순 · 오래된순 · 조회순 · 좋아요순 |
| **찜한 책만** | 내가 북마크한 책만 모아보기 |
| **좋아요 책만** | 내가 좋아요 누른 책만 모아보기 |
| **키워드 검색** | 제목·작가명 실시간 검색 |

적용 중인 필터 개수가 뱃지로 표시되고, 한 번에 초기화 가능

### 🎲 랜덤 추천 (UFO Pick)
- 버튼 클릭 시 UFO가 날아와 책 한 권을 빔으로 들어올리는 인터랙티브 애니메이션
- 랜덤 슬롯머신 효과로 책 제목이 굴러가다가 최종 선택
- 선택된 책의 상세 모달이 자동으로 열림

### 📖 책 상세 모달
- 표지 이미지 · 제목 · 작가 · 장르 · 연령 · 등록일 · 내용 요약
- **좋아요** 💛 · **북마크** 🔖 · **완독 체크** ✅ — 클릭 한 번으로 상태 저장 (localStorage)
- **댓글 기능** — 책마다 코멘트를 남기고 삭제 가능
- **URL 공유** — `#book-{id}` 해시 링크로 특정 책을 바로 공유, 링크 접속 시 모달 자동 오픈
- 모달 내에서 이전/다음 책으로 이동 가능

### 📊 독서 통계
- **완독권 수** · **총 좋아요** · **찜한 책** 요약 카드
- 최근 3개월 완독 현황 — 가로 바 차트
- 장르별 완독 히트맵 — 색상 농도로 독서 취향 시각화
- 통계 카드 클릭 시 해당 필터가 자동 적용되어 목록으로 이동

### 🌙 다크모드 (우주 테마)
- 라이트 모드 : 따뜻한 나무 서재 분위기 (크림·호박색 팔레트)
- 다크 모드 : 심우주 정거장 컨셉 (네이비·스틸 팔레트)
- 다크 모드 전용 별 배경 — 반짝이는 별들, 유성, 행성, 성운 nebula가 캔버스에 동적 생성
- 별들은 레이어별로 다른 드리프트 애니메이션으로 우주 공간감 표현
- 설정이 localStorage에 저장되어 재방문 시 유지

### 🔗 데이터 연동 (Google Sheets CSV)
- Google Sheets를 공개 CSV로 게시하면 별도 백엔드 없이 실시간 연동
- CSV 파싱 엔진 내장 (따옴표 이스케이프, CRLF 등 엣지케이스 처리)
- 책 표지 이미지는 GitHub 저장소의 `bookcovers/` 폴더에 `story_{id}.png` 형식으로 자동 매핑

---

## 📋 데이터 구조 (CSV 컬럼)

Google Sheets에서 아래 헤더를 사용하세요.

| 컬럼명 | 필수 | 설명 |
|---|---|---|
| `id` | ✅ | 책 고유 ID (예: `001`, `002`) |
| `title` | ✅ | 책 제목 |
| `author` | | 작가명 |
| `category` | | 장르 (예: `판타지`, `로맨스`) |
| `age` | | 대상 연령 (예: `유아`, `아동`, `전체`) |
| `date` | | 등록일 `YYYY-MM-DD` |
| `summary` | | 내용 요약 |
| `recommend` | | 추천 여부 (`1` = 추천 컬렉션에 노출) |

> 표지 이미지는 `bookcovers/story_{id}.png` 파일을 저장소에 업로드하면 자동 연결됩니다.

## 🚀 배포 방법

### Vercel (권장)
1. 이 저장소를 Fork 또는 Clone
2. `index.html` 내 `CONFIG.sheetUrl` 을 본인의 Google Sheets CSV 링크로 교체
3. Vercel에 프로젝트 연결 → 자동 배포

### GitHub Pages
1. 저장소 Settings → Pages → Branch: `main` / `root` 선택
2. `https://{username}.github.io/{repo}/` 로 접근
